# SimpleAlias
SimpleAlias is a simple plugin for Minecraft: Java Edition 1.20-1.21.x that provides alias for specific commands.

# Commands
/lobby - Alias for /server lobby.

/hub - Alias for /server lobby.

/survival - Alias for /server survival.

/practice - Alias for /server practice.

/boxpvp - Alias for /server boxpvp.

# Configuration
No configuration needed. Just place this plugin in your plugins folder.
